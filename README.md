# TibSchol Data Explorer

A data exploration proof of concept [static website](https://acdh-oeaw.github.io/tibschol-viz/) for  **TibSchol: The Dawn of Tibetan Buddhist Scholasticism (11th–13th century)**

## 🛠 Tech Stack

- **Next.js** (App Router)
- **React**
- **Tailwind CSS** (custom theme)
- **TypeScript**
- **PostCSS**
- **Google Fonts**:
  - [Noto Serif](https://fonts.google.com/specimen/Noto+Serif) for Latin
  - [Noto Serif Tibetan](https://fonts.google.com/noto/specimen/Noto+Serif+Tibetan) for Tibetan
